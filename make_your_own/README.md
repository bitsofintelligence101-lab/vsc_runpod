# Make Your Own Ollama <-> RunPod Proxy
This how to is a work in progress. Files are provided if you want to diy. it's not overly complicated. basic idea host a model some place, or have the url of a public model.  build the docker image to download from that location. The rest of the docker script would stay the same and is just about integrating with runpod serverless. Then you can use the runpod_deploy.py script in main directory to connect vs code copilot to your model on runpod.
